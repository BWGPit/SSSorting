Insertion sort

Insertion sort is described as follows. Its running time is O(n2) . 

INSERTION-SORT(A)
FOR j = 2 TO A.length DO
    {
        key = A[j];
        i = j-1;
        WHILE i > 0 && A[i] > key DO
            {
                A[i+1] = A[i];
                i--;
            }
        A[i+1] = key;
    }
